Byron UI
Themes

Byron UI for (iOS 13+) It is a friendly UI and different from the default user interface of iOS.

This theme is compatible with some of the system applications.

Features:

  • Phone
  • Mail
  • Photos
  • Notes
  • Safari
  • App Store
  • Music App
  • Flash & Camera (Lockscreen)
  • Keyboard Emoji

Byron UI theme also contains changes for third party applications like:

  • Cydia
  • Zebra
  • WhatsApp (Include UI & Bubbles)

on future updates I plan to add more icons

Extras:

  • Skull UI Switch
  • Emoji Keyboard Icons x3 (Ironman, Batman & Mickey Mouse)

How to apply?

To apply Byron UI the first thing they need is to add the Repo: https://sparkdev.me/repo and once added they must install:

  • Snowboard
  • Snowboard UI Extension

NOTE: trying to install with another theme tools is at your own risk

  • Six new UI switches were added:

    • Alien
    • Flower
    • Sun face
    • Doughnut
    • Snowflake
    • Flag DR (Dominican Republic)

    Byron UI was divided into three sections:

    1. Byron UI
    2. Byron UISwitch
    3. Byron UIApps

    Other Changes:

    • Changes in some icons of the App Store.
    • Added support for the chat bubbles of the messages app.
    • Changes in the WhatsApp settings icon and other icons within the chats.
    • Minor changes to some system icons.
    • Added support for Sileo Package Manager.
    • Minor changes to Zebra and Cydia Package Managers.

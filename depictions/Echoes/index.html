Echoes
Themes

Echoes is Timeloop latest project in terms of iconography and it's mainly focused on blurred glyphs. I've wanted to create an unique and appealing blend between popping colorful gradients and hand-crafted iconography.
The main idea began looking at the current state of iOS Ui interface, which has various blurred elements in within the entire operating system. The blur effect is giving the eye perception of depth from a 2D perspective and plain of view while also maintaining a frosted glass effect.

What it offers

  • The theme currently has 585+ themed icons and also a lot of alternate icons.
  • Works on the iPad too, beautiful and crisp.
  • Every single icon is crafted from ground up trying to create an unique look for each icon.
  • echoes has its own icon mask (as an option) which changes the radius of the icons to a more smooth and rounded shape.
  • Custom settings icons.
  • Notification badge.
  • Respring animation is important, so It has one (spoiler alert! it's yellow)
  • Added 30+ more user requested icons

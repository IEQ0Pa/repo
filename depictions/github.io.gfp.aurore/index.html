Aurore

This tweak replaces the default alarm sounds with music

Currently supports (more coming in the future): Spotify: Playlist, track, or album Apple Music: Any playlist with play & shuffle buttons and radio stations

Notable Features:

  • Fully integrated into clock app
  • Shuffle or linear playback modes
  • Customizable volume fades
  • Customizable snooze options
  • Buttons positions are configurable
  • Weather summary on dismiss
  • Bluetooth and AirPlay support
  • Run shortcuts on fire and dismiss
  • Backup alarms are created in case of playback failure or internet issues
  • In the event that iOS kills the music application during a long snooze, Aurore will now relaunch the application and restart the music
  • The interval between the volume increases has been halved for a smoother effect
  • Decimal values for fade in time and snooze time now supported.
  • Apple Music delay increased to accommodate slower loading playlists
  • Added an option to kill music application before launching music

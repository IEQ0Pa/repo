Accent Cards

An Awesome Card Style Xen HTML Widget

What's this?

Accent Cards is an OPW Xen HTML Widget. OPW stands for One Page Widget, which is an all in one Widget that gives you a Setup completely Ready out of the box for your beautiful Homescreen. OPWs already provides a Dock, and set of App shortcuts that you can easily place your own. Accent Cards also provide a set of Tiles stored in each Card. These tiles can be easily modified by enabling Edit Mode through the settings.

Detailed Feature List -

Card Wallpaper Illusion -

The Centre Part of the Widget, creates an illusion that the Wallpaper itself is a Card, and that card is above the Screen. This is a unique style done entirely with CSS.

Cards -

Accent Cards include a set of 3 Cards that can be scrolled, and opened to see what each Card has inside. Each Card Opener would show a set of Tiles that can be used as App shortcuts or display Important Information, like Weather or a Music Player.

Customization -

Accent Cards has a full featured Settings panel, that can help you customize the Widget per your Desire. You can also change the Accent Colors of the Widget with the help of a Color Picker.

Edit Mode -

This is a special Setting that can be enabled from the Settings Panel. With this, you can easily modify the Positions as well as the Sizes of the Tiles inside each Card Opener. After enabling Edit mode from the Settings Panel, to change the position of a tile, simply tap and drag the Tiles to your desired Position. To change the Size, thanks to @junesiphone's taphold js Library, you can tap and hold on the Tile to open Settings to change the size or reset everything. Make sure to disable the Setting from the Settings Panel once done.

App Shortcuts -

Being an OPW, app shortcuts are a must. This widget provides that, along with the ability to change those apps as per your desire. Thanks to @junesiphone's taphold js Library, you can tap and hold on the App shortcuts in the Dock, or the tiles to change an App. An App Drawer would open up where you can choose your Desired App. There'll be Blank apps with a white background, those are Placeholders, you need to change them to your desired App.

Some Instructions/Guides/Hints -

Applying the Widget -

To Apply the Widget there are two methods, first decide where you wanna apply it, Background or Foreground?
If you place it in the Background the Widget will remain fixed in the Homescreen even if you swipe through the Homescreen Pages. For Background, go to Settings -> XenHTML -> Homescreen -> Background -> Add Widget -> Accent Cards
If you place it in the Foreground, the widget would remain only on the Page you applied the Widget to. For Foreground, go to your Homescreen -> Enable Wiggle Mode -> Tap on the Add Widget Button at the bottom of the Homescreen and Apply it.

Keeping the Homescreen Blank -

An OPW is a full page widget, which means you don't need to use the Homescreen icons anymore. So, it is recommended, to either hide all the icons as well as the Dock, or just keep the First Page of the Homescreen completely Blank. As you all know, iOS has a native feature, that you can always keep your First Page of the Homescreen completely blank, for that you just need to shift all the Icons in the first page to the Second Page. And that's it.

Touch Input related Problems (Can't touch on the Widgets) -

There are some tweaks that cause problem with touch input on the Widgets. Other than that, the problem also arises from your XenHTML Settings.
Make sure in the XenHTML Settings, if you applied the widget to Background, you've got Enabled Allow Touch on Background Toggle from the Homescreen XenHTML Settings also make sure you've got no apps above the Widget in the Homescreen.
If you applied the Widget to Foreground, make sure you've got Prevent Touch Behind Settings disabled from XenHTML Homescreen Settings in the Foreground Settings Section also make sure you've got no apps above the Widget in the Homescreen.

Clay CC
Clay CC is Complete Premium set for icons .
Features :
• 50+ Premium icons
• 9 Magma Evo setups (2 Special For Clay CC)
• Apple Stock CC toggles
• CCModules
• Animated toggles
Clay CC Supports :
• Prysm Control Center
• Stock Apple’s CC Control Center
• Veza Toggles +
• SystemWide Music Control’s
• Small & Large icons Size options for Toggles
Manual important Part of Use :
• After Installing ClayCC
• For Prysm you need Go in this Folder location using Filza /Library/Themes/ClayCC Large.theme Or ClayCCSmall.theme/Bundle/Prysm inside this you’ll find Icons folder Copy it and Paste inside /Library/Application Support/Prysm/
• Adjust toggles from Control Center settings which is with black background icons.
• Apply ClayCC using Snowboard or iThemer.
• Now Respring your device.
• For Apple Stock Control Center you need Go in this Folder location using Filza /Library/Themes/ClayCC Large.themeOr Small.theme/Bundles/BetterCCIconsPro and Copy Wifi& Bluetooth files & Paste inside /Library/PreferenceBundles/BetterCCIcons13.bundle Then enable Glyphs Mode in the settings.
• From Directory I Provided You can Choose Small or LargeClayCC version Which one you Like.
• You Can also choose from Arc CC Small & Large version.
Make Sure All Below Tweaks Installed :
• Snowboard or iThemer
• Snowboard UI Extention ( If You Using Snowboard )
• Flipconvert from this Repo - https://julioverne.github.io/
• Flipswitch from this Repo - https://rpetri.ch/repo/
• BetterCCIconsPro ( if You want to use ClayCC with Apple Stock CC then WiFi and Bluetooth theme with this tweak )
